Over Mij

Ik ben Vishal Tewari, BBL-student Software Developer aan het Techniek College Rotterdam. Naast mijn opleiding werk ik bij Stapelzorg, waar ik mijn programmeervaardigheden verder ontwikkel in een echte werkomgeving.

Op dit moment werk ik aan projecten zoals mijn eigen InvoiceApp, waar ik gebruik maak van C#, Entity Framework en SQLite. Ik leg de nadruk op structuur, validatie en schone code die goed leesbaar blijft — iets wat ik belangrijk vind in elk project.

Mijn interesse ligt bij software die niet alleen goed werkt, maar ook logisch is opgebouwd. Ik denk graag in oplossingen, niet in problemen.

Uiteindelijk wil ik doorgroeien tot een full-stack developer die complete applicaties kan ontwerpen, bouwen en optimaliseren — van database tot front-end, met oog voor detail en gebruiksgemak.
