Portfolio – Vishal Tewari

BBL-student Software Developer – Techniek College Rotterdam

Dit project is mijn persoonlijke portfolio website.
Het doel van deze site is om mijn werkervaring, opleiding, projecten en vaardigheden duidelijk en professioneel te presenteren voor school, stage en toekomstige werkgevers.

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Inhoud van de Website:
```
De website bestaat uit 8 pagina’s:
	1.	Home – introductie met typing-animatie
	2.	Over Mij – achtergrond, hobby’s en informatiekaart
	3.	Opleidingen & Certificaten – opleidingen, cursussen en certificaten
	4.	Werkervaring – overzicht in tijdlijnvorm
	5.	Skills – technische vaardigheden
	6.	Projecten – projecten met screenshots
	7.	Contact – contactformulier
	8.	Bedankt – bevestiging na verzenden formulier
```
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Gebruikte Technieken:
```
	•	HTML5
	•	CSS3 (volledig gevalideerd)
	•	JavaScript (navigatie, lightbox, typing-effect)
	•	Responsive design (flexbox + grid)
	•	GitHub versiebeheer
```
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

2. Structuur Begrijpen

Alle pagina’s zijn handmatig gekoppeld met links.
Belangrijkste locaties:
```
	•	HTML-bestanden → /Pages/
	•	Stijlen → /css/style.css
	•	JavaScript → /js/script.js
	•	Afbeeldingen → /images/
```
  ⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
  
  3. Aanpassingen Doen

Wil je de inhoud wijzigen:
```
	•	Teksten aanpassen → open het juiste HTML-bestand in /Pages/
	•	Kleuren of layout wijzigen → pas variabelen of secties aan in css/style.css
	•	Navigatie wijzigen → wijzig de <nav>-links in elk HTML-bestand
	•	Afbeeldingen vervangen → plaats nieuwe afbeeldingen in /images/ en update de src-attributen
```
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
